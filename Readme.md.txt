# 📰 AI Fake News Detector (Desktop App)

## 🚀 Overview
A professional Python-based desktop application that detects fake news using Machine Learning.

## ✨ Features
- Dark mode UI
- Real/Fake prediction
- Confidence score
- Prediction history
- Clean interface

## 🛠️ Tech Stack
- Python
- Tkinter
- Scikit-learn
- Pandas

## ▶️ How to Run

pip install -r requirements.txt
python app.py

## 📸 Output
Enter news → Click Analyze → Get result instantly
