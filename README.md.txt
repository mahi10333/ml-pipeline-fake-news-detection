# 📰 Fake News Detection System

## 📌 Overview
This project uses Machine Learning and NLP to classify news as REAL or FAKE.

## 🚀 Features
- Text preprocessing using NLP
- TF-IDF vectorization
- Logistic Regression model
- CLI + Web API support

## 🛠️ Tech Stack
- Python
- scikit-learn
- NLTK
- Flask

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Train model
python src/train.py

### 3. Run main program
python main.py

### 4. Run web app
python app.py

## 📊 Dataset
Custom dataset included in `/data/news.csv`
